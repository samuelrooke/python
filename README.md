# Python Coursework - HAMK & TAMK

Python exercises completed during Business Information Systems studies at HAMK University of Applied Sciences and TAMK Tampere University of Applied Sciences. Covers core Python from syntax basics through object-oriented programming, data analysis, and a text-based game project.

**TT** = in-class exercise · **KT** = homework assignment

---

## Contents

| Folder | Topic | Highlights |
|--------|-------|------------|
| `/1/` | Basics — input, variables, f-strings | Electricity bill calculator |
| `/2/` | Strings — slicing, indexing, built-ins | DNA nucleotide counter |
| `/3/` | Arithmetic & comparison | Petrol vs diesel cost comparison |
| `/4/` | Loops — `while` / `for` | Ski-jump scoring system |
| `/5/` | Functions & validation | Temperature converter (°C / °F / K) |
| `/6/` | Algorithms | Palindrome checker (punctuation-aware) |

| `/9/` | Object-oriented programming | `Pizza` and `Pelaaja` (Player) classes |
| `/10/` | JSON & modules | Location loader with custom class |
| `Analysoinnin-Perusteet-HAMK/` | Data analysis | pandas, matplotlib, seaborn |
